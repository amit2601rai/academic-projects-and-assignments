           
        The project "File Sharing System" has following features:
         
         1.user registeration :- any user can register on this website after filling the valid information.
         
         2.login :- any registered user can login with help of username and password.
         
         3.upload file :- user can upload their files.
         
         4.download file :- user can download own files as well as download other's files on which they got permission.
         
         5.share file :- user can share file to other users.
         
         6.remove file :- user can remove its own uploaded file.
